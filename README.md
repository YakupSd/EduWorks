# 🎓 Classroom Quiz Arena – Öğretmen & Öğrenci Etkileşimli Bilgi Yarışması

**Classroom Quiz Arena**, öğretmenlerin ders sırasında öğrencilerle gerçek zamanlı etkileşim kurmasını sağlayan, eğlenceli ve rekabetçi bir **bilgi yarışması platformudur**.  
Bu proje, sınıf içi katılımı artırmak, öğrenmeyi oyunlaştırmak ve öğrencilerin motivasyonunu yükseltmek amacıyla geliştirilmiştir.

---

## 🚀 Özellikler

- 👨‍🏫 **Öğretmen Paneli**
  - Soru oluşturma ve yönetme  
  - Oyun başlatma ve kontrol etme  
  - Anlık skor takibi  

- 👧🧒 **Öğrenci Katılımı**
  - Gerçek zamanlı soru cevaplama  
  - Puan ve sıralama sistemi  
  - Eğlenceli yarışma deneyimi  

- 🤖 **AI Destekli Soru Üretimi**
  - Gemini API ile otomatik soru üretimi  
  - Ders konusuna göre dinamik quiz oluşturma  

- ⚡ **Gerçek Zamanlı Etkileşim**
  - Canlı skor tablosu  
  - Öğrenci–öğretmen anlık geri bildirim  

---

## 🛠️ Teknolojiler

- **Node.js**
- **React / Next.js (AI Studio template)**
- **Gemini API**
- **Tailwind CSS**
- **WebSocket / Real-time features (opsiyonel)**

## ⚙️ Localde Çalıştırma

### Gereksinimler
- Node.js (v18+ önerilir)

### Kurulum

```bash
# Bağımlılıkları yükle
npm install

# .env.local dosyasına Gemini API Key ekle
GEMINI_API_KEY=YOUR_API_KEY_HERE

# Projeyi çalıştır
npm run dev
